Mouse Movement Path Visualizations

About These Visualizations

These visualizations show different mouse movement models used to generate natural-looking cursor movements between points. Each model has distinct characteristics that make it suitable for different types of movements.

Bezier - Standard curved path
BezierHigh - More complex curve with jitter
Physics - Spring-mass-damper simulation
PhysicsLowDamping - Oscillatory physics movement
MinimumJerk - Biomechanically accurate

Implementation Details

Bézier Curves: Uses cubic Bézier curves with control points calculated based on the distance between start and end points. The control points are placed asymmetrically to create natural-looking curved trajectories.

Physics Simulation: Models the cursor as a mass connected to the target by a spring with damping. This creates realistic acceleration, deceleration, and potentially overshoot behavior.

Minimum Jerk Trajectory: Based on research showing that human movements minimize the rate of change of acceleration (jerk). This produces very smooth paths that closely mimic natural human movements, especially for direct targeting tasks.

Jitter: The Ornstein-Uhlenbeck process is used to add realistic tremor/jitter, simulating the slight imprecision in human movements. This stochastic process creates correlated noise that appears more natural than random perturbations.
